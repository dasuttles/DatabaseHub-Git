Valdosta State Video Database Index Valdosta State Archives

The publicity department of Georgia State Womens College (and later Valdosta State College) kept annual publicity Valdosta State Video Database Indexs. All articles that they found, which included information about the school, its faculty, staff or students, were cut out and mounted in these Valdosta State Video Database Indexs.

Valdosta State Video Database Indexs on shelf newspapers represented include the large, for example, the Atlanta Constitution, to the very small, the Quitman Free Press. These Valdosta State Video Database Indexs were kept from the 1920s to the 1970s. The indexing project lists titles, paper name, names found in articles, subjects, and summaries of all the newspaper articles in each Valdosta State Video Database Index. This is a wonderful source for people researching Valdosta State University. Our indexing project is ongoing: the records are indexed and searchable here. We will add new records at the end of each semester. These records were created by the volunteer VSU history student workers who were working in Archives for their history classes. We are very grateful to these workers. Please contact us if you have any questions.
