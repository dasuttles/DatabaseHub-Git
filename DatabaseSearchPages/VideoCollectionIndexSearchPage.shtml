The video collection indexing project lists titles, event names, names found in videos, subjects, and summaries of each video. This is a wonderful source for people researching Valdosta State University. Our indexing project is ongoing. We will add new records at the end of each semester. These records were created by the volunteer VSU history student workers who were working in Archives for their history classes. We are very grateful to these workers. Please contact us if you have any questions or find something you would like digitized.
